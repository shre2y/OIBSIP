# Task 3 – Temperature Converter Website

This project is a **temperature converter website** built using **HTML, CSS, and JavaScript**.

## Features
- Convert temperature between:
  - Celsius (°C)
  - Fahrenheit (°F)
  - Kelvin (K)
- Input validation to ensure numeric values
- Clean and modern user interface
- Instant conversion on button click

## Technologies Used
- HTML
- CSS
- JavaScript
- GitHub for version control
